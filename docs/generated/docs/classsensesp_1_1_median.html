Detailed Description

Outputs the median value of sample_size inputs.

Creates a sorted list of sample_size values and outputs the one in the middle (i.e. element number 'sample_size / 2').

Parameters
sample_sizeNumber of values you want to take the median of.
config_pathPath to configure this transform in the Config UI.
