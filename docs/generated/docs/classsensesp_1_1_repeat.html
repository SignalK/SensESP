Detailed Description

template<typename FROM, typename TO>
class sensesp::Repeat< FROM, TO >

Repeat the input at specific intervals.

Ensures that values that do not change frequently are still reported at a specified interval. If the value has not changed in interval milliseconds, the current value is emmitted again.

The repetition only occurs if the value has changed.

Parameters
intervalMaximum time, in ms, before the previous value is emitted again.
config_pathPath to configure this transform in the Config UI.
