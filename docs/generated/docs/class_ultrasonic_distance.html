Detailed Description

Reads a DYP-A02YYWM-V1.0 (and possibly other types) ultrasonic sensor and outputs the number of microseconds that it takes the sound wave to bounce off an object and return to the sensor.

Output must be sent to a Linear Transform to convert to a distance.

See also
https://github.com/SignalK/SensESP/tree/master/examples/ultrasonic_level_sensor
Parameters
trigger_pinThe pin that is set to HIGH/LOW to start/stop the measurement.
input_pinThe pin that reads the measurement
read_delayThe time between reads of the sensor, in ms. Defaults to 1000.
config_pathThe path to configure read_delay in the Config UI.
